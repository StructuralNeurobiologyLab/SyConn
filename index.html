SyConn

Automated synaptic connectivity inference
for volume electron microscopy

  • GitHub Repository
  • Sample data with code snapshot
  • Documentation

Teravoxel volume electron microscopy datasets from neural tissue can now be acquired in weeks, but data analysis requires years of manual labor (see also Fig. 1). The SyConn framework uses deep convolutional neural networks (CNN) and random forest classifiers (RFC) to infer a richly-annotated synaptic connectivity matrix from manual neurite skeleton reconstructions by automatically identifying mitochondria, synapses and their types, axons, dendrites, spines, myelin, somata and cell types.

SyConn was developed in Prof. Winfried Denk’s department Electrons - Photons - Neurons at the Max Planck Institute of Neurobiolgy in Martinsried, Germany.

Getting Started

Python 2.7 is required (we recommend Anaconda). SyConn has been tested on Linux distributions so far (CentOS and Arch Linux).
